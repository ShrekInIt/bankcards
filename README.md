Сервис для управления банковскими картами: аутентификация пользователей, просмотр карт, переводы, администрирование пользователей и карт, обработка заявок на блокировку

## Стек технологий

- Java 21
- Spring Boot 3.5.10
- Spring Security
- Spring Data JPA
- Liquibase
- PostgreSQL
- Maven
- Docker / Docker Compose
